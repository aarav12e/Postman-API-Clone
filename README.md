# Postman API Clone
[Live Demo](https://postman-api-clone.vercel.app/) · Demo site: `https://postman-api-clone.vercel.app/`

A lightweight Postman-like API client built with modern web tech — send requests (GET, POST), add headers & body, and inspect responses. Designed for learning, demos and quick API testing.

---

## 🚀 Live Demo
Try it now: https://postman-api-clone.vercel.app/

---

## ✨ Features
- Send HTTP requests: **GET**, **POST**, **PUT**, **DELETE**  
- Set request URL, headers and request body (raw JSON/text)  
- Pretty-print JSON responses and view raw response  
- Shows response status code and response time  
- Simple, responsive UI — great for demos and learning

## 💻 Local Setup

**1. Clone**
```bash
git clone <your-repo-url>
cd <your-repo-folder>
```

**2. Install dependencies**
```bash
npm install
```

**3. Run locally**
```bash
npm run dev
```

Then open `http://localhost:5173` in your browser.
